## Pitfalls
1. **Paynow authemail mismatch** — wasted time debugging why test payments failed. The test integration (23997) requires tatendawalter62@gmail.com, not the user's email. Fix: PAYNOW_MERCHANT_EMAIL env var
2. **Stale pending payments** — first payment attempt creates a pending record, retry was blocked. Fix: delete stale pending on retry
3. **Browser form submission showed raw response** — form.submit() navigated to Paynow's URL-encoded response text. Fix: use fetch() instead, parse response, redirect to browserurl
4. **Background agents couldn't access Bash** — worktree isolation blocked shell commands. Had to build both branches manually
5. **All auctions were expired** — ended them for testing, then had no active listings for the demo. Fix: always create fresh demo listings after testing
