Technical Architecture

FRONTEND

React 18 + TypeScript

  • Vite build, Tailwind CSS
  • shadcn/ui (48+ components)
  • TanStack Query (data fetching)
  • Zustand (auth state)
  • Lazy-loaded routes
  • Mobile-first, WCAG 2.1 AA
BACKEND

Supabase

  • PostgreSQL + RLS (Grade A security)
  • 18 Edge Functions (Deno)
  • Auth with JWT
  • Realtime subscriptions
  • Storage (images, stock cards)
  • Atomic RPC functions
PAYMENTS

Multi-Provider

  • Paynow Gateway (primary)
  • EcoCash express checkout
  • OneMoney express checkout
  • Stripe (card fallback)
  • Cloudflare bypass pattern
  • Dual webhook handler
